# 助手与记忆

## 能力

- 助手名称、头像、人设、模型参数、背景、预设消息和快捷消息。
- 助手级/全局长期记忆，近期聊天引用。
- 模式注入、Lorebook、正则输入输出处理、技能、MCP 和本地工具选择。
- 会话可按助手授权覆盖系统提示词和提示词注入。
- 助手默认启用匿名提问箱，可在“助手设置 → Local Tools（本地工具）”中关闭。

## 数据

- 助手配置序列化在 `Settings.assistants`。
- 长期记忆存 Room：`MemoryEntity`/`MemoryDAO`。
- 会话独立提示词随 Conversation 保存。

## 核心入口

- 模型：`data/model/Assistant.kt`
- UI：`ui/pages/assistant/`、`ui/pages/assistant/detail/`
- 记忆：`data/repository/MemoryRepository.kt`、`data/ai/tools/MemoryTools.kt`
- 技能：`data/files/SkillManager.kt`、`ui/pages/extensions/SkillsPage.kt`

## 修改提示

- 新助手字段必须给默认值并考虑 Settings JSON 兼容。
- 人设相关行为同时检查助手 system prompt 和 conversation system prompt。
- 记忆范围由 `enableMemory`、`useGlobalMemory` 控制，不能混淆助手隔离与全局共享。
