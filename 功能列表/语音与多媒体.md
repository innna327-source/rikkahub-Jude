# 语音与多媒体

## 能力

- TTS 多供应商、段落朗读、自动播放、仅引用/仅英文等策略。
- ASR 提供商和系统输入法语音通话界面。
- 助手可在启用本地工具后主动邀请当前聊天用户通话；来电支持震动、滑动接听、拒接和 30 秒未接反馈。
- 图片、PDF、DOCX、PPTX、EPUB 等多模态输入；OCR 可使用独立接口。
- 图片生成、文件预览、本地图片持久化和 Markdown/代码/公式渲染。

## 核心入口

- 语音模块：`speech/src/main/java/me/rerere/tts/`、`me/rerere/asr/`
- 通话 UI：`ui/pages/chat/VoiceCallOverlay.kt`
- 来电 UI：`ui/pages/chat/IncomingVoiceCallOverlay.kt`
- 自动朗读：`ui/pages/chat/TTSAutoPlay.kt`
- 文档解析：`document/`
- OCR：`data/ai/transformers/OcrTransformer.kt`、`data/ai/prompts/OcrPrompt.kt`
- 图片生成：`ui/pages/imggen/`、`GenMediaRepository.kt`
- 文件：`data/files/FilesManager.kt`

## 修改提示

- 键盘关闭与挂断是不同生命周期事件，修改通话 Dialog 时不可混用。
- 主动来电复用工具审批状态；接听后以 `VoiceCall` 模式续跑，拒接或超时结果必须返回模型。
- 多媒体 URI 写入持久化数据前要复制到应用私有目录。
- OCR/压缩独立 API 配置不能污染普通聊天供应商设置。
